# Diabetes-Classification-Project
1. **Overview:** This project predicts diabetes using a Random Forest Classifier with a remarkable 98% accuracy on the Diabetes prediction dataset.
  
2. **Preprocessing:** Null values were addressed through imputation, class imbalance handled, and features encoded and scaled for model stability.

3. **Modeling:** Implemented a Random Forest Classifier, achieving an accuracy of 98% on the test set.

4. **Results:** The model's performance metrics include accuracy of 98%, showcasing its effectiveness in predicting diabetes.
